# Control de Gastos

Aplicación web desarrollada con React y TypeScript para la administración de gastos personales. Permite definir un presupuesto, registrar gastos por categoría, editar y eliminar registros, visualizar el presupuesto disponible y realizar seguimiento financiero de forma sencilla e intuitiva.

## Funcionalidades

- Registro de presupuesto inicial.
- Alta, edición y eliminación de gastos.
- Clasificación de gastos por categorías.
- Cálculo automático del presupuesto disponible.
- Indicadores visuales del estado financiero.
- Filtrado de gastos por categoría.
- Persistencia de datos mediante LocalStorage.
- Interfaz responsiva y amigable.

## Tecnologías utilizadas

- React
- TypeScript
- Context API
- useReducer
- Tailwind CSS
- LocalStorage
- React DatePicker
- UUID

Este proyecto fue desarrollado para fortalecer conocimientos en gestión global de estado con Context API y useReducer, tipado con TypeScript, manejo de formularios y desarrollo de interfaces modernas con React.
